VSL Player Examples

1. Full Featured Player

All controls enabled - seeking, volume, and fullscreen available.

VSLPlayer.init("#player-1", {
  videoSrc: "https://vz-70795825-f8d.b-cdn.net/b271f9e5-27f3-4249-ac3a-14360eda1b82/playlist.m3u8",
  colorTheme: "#5A3FFF",
  hideControls: false,
  volumeControl: true,
  fullscreenControl: true,
  saveProgress: true,
  distortionFactor: 0.4,
  defaultQuality: "high"
});

2. VSL Mode (No Seeking)

Perfect for sales videos - users can play/pause and adjust volume, but cannot skip ahead.

VSLPlayer.init("#player-2", {
  videoSrc: "https://vz-70795825-f8d.b-cdn.net/b271f9e5-27f3-4249-ac3a-14360eda1b82/playlist.m3u8",
  colorTheme: "#00C853",
  hideControls: true,
  volumeControl: true,
  fullscreenControl: false,
  saveProgress: true,
  distortionFactor: 0.3,
  defaultQuality: 720
});

3. Autoplay with Muted Overlay

Starts playing automatically (muted) with an overlay prompting user to unmute.

VSLPlayer.init("#player-3", {
  videoSrc: "https://vz-70795825-f8d.b-cdn.net/b271f9e5-27f3-4249-ac3a-14360eda1b82/playlist.m3u8",
  colorTheme: "#FF1744",
  autoplay: true,
  hideControls: true,
  volumeControl: true,
  saveProgress: true,
  defaultQuality: "medium"
});

4. Minimal Player (Play/Pause Only)

Most restrictive - only play and pause controls, no volume or fullscreen.

VSLPlayer.init("#player-4", {
  videoSrc: "https://vz-70795825-f8d.b-cdn.net/b271f9e5-27f3-4249-ac3a-14360eda1b82/playlist.m3u8",
  colorTheme: "#0080FF",
  hideControls: true,
  volumeControl: false,
  fullscreenControl: false,
  saveProgress: false,
  defaultQuality: "auto"
});
